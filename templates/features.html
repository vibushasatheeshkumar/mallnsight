PLATFORM CAPABILITIES

Everything You Need for Static Analysis

MallnSight inspects suspicious files completely offline — no code is ever executed. Every capability below runs locally in your own environment.

Hash Analysis

Generates MD5, SHA1 and SHA256 hashes for unique file fingerprinting, useful for threat-intel lookups and de-duplication.

File Metadata

Extracts filename, size, extension and MIME type to quickly characterize an unknown sample.

PE Analysis

Parses Windows PE headers — architecture, compile timestamp, entry point, image base, sections, imports and exports — via pefile.

Entropy Analysis

Calculates whole-file and per-section Shannon entropy to flag packed, encrypted or obfuscated binaries.

String Extraction

Pulls ASCII and UTF-16LE strings and automatically flags suspicious indicators — injection APIs, command interpreters, registry persistence, URLs, IPs and anti-debug checks.

YARA Detection

Scans against a bundled, extensible YARA rule set covering process injection, dynamic API resolution, command execution and packing.

Risk Scoring

Combines every signal above into a single 0–100 risk score and a clear verdict: Clean, Low Risk, Suspicious or High Risk.

PDF Investigation Report

One-click export of the full findings into a shareable PDF report, generated with reportlab.

Fully Offline

Nothing is uploaded to a third party. Files, hashes and results stay on the machine running MallnSight.
